I applied Double Machine Learning techniques in Python to study the causal relationship between corruption and firm growth in Vietnam. I used pandas, NumPy, and pyjanitor to clean, merge, and harmonize multiple survey and administrative data sources into a panel dataset and to construct regression-ready variables. Estimation was carried out using  DML and CausalForestDML estimators, combined with scikit-learn models such as LogisticRegression, RandomForestClassifier, and RandomForestRegressor.
